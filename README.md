# C_Compiler

## 概要

C++で書いた自作C言語コンパイラです。  
[低レイヤを知りたい人のためのCコンパイラ作成入門](https://www.sigbus.info/compilerbook)をもとにインクリメンタルに機能を追加していきます  
参考サイトと異なる点は以下の通りです  
- C++で書かれている
- Visual Studioを使用し、テストにはGoogleTestを用いている
